What is this?

Stream Guess is a simple game where you are given a clip from popular streamers and you need to guess which streamer it is from the options provided.


Stream Guess not only provides a way for stream fans to test their knowledge, but it allows you to discover new streamers too!

How is this made?

This site is written using Angular 4, HTML5, and CSS3 with a REST API backend providing the data.


Stream clips and data are being queried from the Twitch API and stored on server. The clips are being constructed from data parsed from the API because I wanted to display the raw clips without any context to provide a more enganging and fun experience that was harder to guess.
